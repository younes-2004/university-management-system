# University Management System

Système de gestion universitaire complet.

## Structure du Projet

- `backend/` - API REST Spring Boot
- `frontend/` - Interface utilisateur JavaScript vanilla

## Technologies

### Backend
- Java 22
- Spring Boot 3.3.x
- MySQL
- JWT Authentication

### Frontend
- HTML5
- CSS3
- JavaScript (Vanilla)

## Installation

### Prérequis
- Java 22
- MySQL 8+
- Maven 3.8+
- Git

### Backend Setup

```bash
cd backend
mvn clean install
mvn spring-boot:run
